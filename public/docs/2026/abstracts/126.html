В современных исследованиях свойств и состава угля особую актуальность приобретает разработка методов получения модельных веществ, воспроизводящих золы уноса. Данная работа посвящена изучению способов создания достаточно однородных и стабильных веществ, имитирующих состав отходов добычи и переработки углей. Для разработки выбраны пробы углей: каменный, бурый, антрацит. Процесс подготовки включал стадии кондиционирования исходного материала, гомогенизацию, последовательное озоление в контролируемых условиях в муфельной печи. Основной задачей исследования было выявление участков с равномерным выделением воды и оксидов углерода, характеризующимися плато на кривой потери массы при проведении исследования методом термогравиметрического анализа с масс-спектрометрическим детектированием. На рисунке представлен пример термограммы, полученной по результатам последовательного озоления пробы антрацита. Выделен интервал температур, при которых не происходит пикового выделения исследуемых элементов.

Пример термограммы, полученной по результатам озоления антрацита

Следует выделить две основные особенности подготовки модельных веществ золы углей, имитирующих различные этапы сжигания: необходимо выбирать низкозольные угли, температура и время озоления устанавливается экспериментально для каждого образца углей. Полученные данные имеют важное практическое значение для изучения процессов сжигания углей. Предложенный подход позволяет воспроизводить золы уноса с высокой степенью точности, что открывает перспективы для создания стандартных образцов.

Работа выполнена в рамках Программы стратегического академического лидерства «Приоритет-2030».
